    Alwyn Voorhoeve's Portfolio website

    I am Alwyn Voorhoeve and i am a former ICT student from HZ University of Applied Sciences in Middelburg. I had to make this website during my time there and I have now updated the contents a bit so i can show it in my application to Creative Business at BUas. This is the Homepage of my old portfoliowebsite.

    This is the html version of this website since i can't host the laravel/php version on github pages. If i wanted to include that in my resume I would've had to pay for some hosting service so i'm using github. There were a bunch of features I implemented like the ability to create an account, log in with it and i had a whole forum on the blog page where you could post stuff. I've left the website its original state.

    I made this website a couple years ago as a project for my ICT studies. The dashboard shows my study progress and the courses I have taken in my first year there, the FAQ page contains answers to common questions about the college itself and the blog page contains some feedback i recieved on the website and a SWOT analysis i had to make about myself.

    Tot nu toe heb ik het erg naar mijn zin op deze opleiding en ben ik overtuigd dat ik een goede studiekeuze heb gemaakt. Ik heb vaak last van uitselgedrag en word vaak afgeleid van dingen als ik ze niet interresant vind, maar dat heb ik bij deze opleiding niet. De stof is lastig, maar het lukt bijna altijd om eruit te komen. En ik hou ook zelf wel van een uitdaging.

    Waarom is ICT de juiste opleiding voor mij?

      -Ik vind het leuk om met computers te werken
      -Ik heb ervaring nodig met programmeren als ik ooit een gamedesigner wil worden
      -Ik hou van games spelen en ik wil meer leren over hoe ze gemaakt worden.
